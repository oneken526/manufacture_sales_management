## タスク概要

得意先マスタの CRUD 操作を提供する `CustomerController` とモデルを実装する。`Customer` モデルは SoftDeletes・締日（closing_day）・与信限度額（credit_limit）を持ち、一覧・詳細・作成・編集・論理削除の各画面を提供する。得意先別特別単価（`CustomerSpecialPrice`）のモデルとリレーションも合わせて実装する。

## 実装詳細

### 1. Customer モデル 🔵

**信頼性**: 🔵 *DBスキーマ・要件定義 REQ-005 より*

**実装ファイル**: `app/Models/Customer.php`

```php
class Customer extends Model {
    use SoftDeletes, HasFactory;
    protected $fillable = [
        'name', 'name_kana', 'postal_code', 'address',
        'phone', 'email', 'closing_day', 'credit_limit', 'notes',
    ];
    public function specialPrices() { return $this->hasMany(CustomerSpecialPrice::class); }
    public function quotations()     { return $this->hasMany(Quotation::class); }
    public function orders()         { return $this->hasMany(Order::class); }
    public function invoices()       { return $this->hasMany(Invoice::class); }
}
```

### 2. CustomerSpecialPrice モデル 🔵

**信頼性**: 🔵 *DBスキーマ・要件定義 REQ-006 より*

**実装ファイル**: `app/Models/CustomerSpecialPrice.php`

```php
class CustomerSpecialPrice extends Model {
    use SoftDeletes;
    protected $fillable = ['customer_id', 'product_id', 'unit_price'];
    public function customer() { return $this->belongsTo(Customer::class); }
    public function product()  { return $this->belongsTo(Product::class); }
}
```

### 3. CustomerController 🔵

**信頼性**: 🔵 *Webルート仕様 api-endpoints.md より*

**実装ファイル**: `app/Http/Controllers/CustomerController.php`

- `index()`: 名前・フリガナ検索 + ページネーション（20件/page）
- `store()`: バリデーション（name 必須, closing_day: 1〜31 or 月末, credit_limit: 0以上）
- `update()`: 同上
- `destroy()`: SoftDelete

### 4. ルート設定 🔵

**信頼性**: 🔵 *Webルート仕様より*

```php
Route::middleware(['auth', 'role:admin|sales'])->group(function () {
    Route::resource('customers', CustomerController::class);
});
```
