## タスク概要

商品カテゴリ（`ProductCategory`）と商品（`Product`）のマスタ管理 CRUD を実装する。商品カテゴリは入れ子（親カテゴリ）に対応し、商品は単価・カテゴリ・品番を持つ。見積・受注の明細行で商品を選択する際の Ajax 検索エンドポイント（`GET /api/products/search`）も実装する。

## 実装詳細

### 1. ProductCategory モデル 🔵

**信頼性**: 🔵 *DBスキーマ・要件定義 REQ-007 より*

**実装ファイル**: `app/Models/ProductCategory.php`

```php
class ProductCategory extends Model {
    use SoftDeletes, HasFactory;
    protected $fillable = ['name', 'parent_id', 'sort_order'];
    public function parent()   { return $this->belongsTo(ProductCategory::class, 'parent_id'); }
    public function children() { return $this->hasMany(ProductCategory::class, 'parent_id'); }
    public function products() { return $this->hasMany(Product::class, 'category_id'); }
}
```

### 2. Product モデル 🔵

**信頼性**: 🔵 *DBスキーマ・要件定義 REQ-008 より*

**実装ファイル**: `app/Models/Product.php`

```php
class Product extends Model {
    use SoftDeletes, HasFactory;
    protected $fillable = ['category_id', 'code', 'name', 'unit', 'unit_price', 'notes'];
    public function category()       { return $this->belongsTo(ProductCategory::class, 'category_id'); }
    public function specialPrices()  { return $this->hasMany(CustomerSpecialPrice::class); }
    public function quotationDetails(){ return $this->hasMany(QuotationDetail::class); }
    public function stockLots()      { return $this->hasMany(StockLot::class); }
}
```

### 3. ProductController 🔵

**信頼性**: 🔵 *Webルート仕様より*
