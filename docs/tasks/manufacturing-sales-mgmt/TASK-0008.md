## タスク概要

倉庫マスタの CRUD 操作を提供する `WarehouseController` と `Warehouse` モデルを実装する。倉庫は在庫管理（stock_quantities）のキーとなるエンティティであり、名称・コード・メモを持つ。在庫残があっても論理削除は可能だが、警告を表示する。

## 実装詳細

### 1. Warehouse モデル 🔵

**信頼性**: 🔵 *DBスキーマ・要件定義 REQ-009 より*

**実装ファイル**: `app/Models/Warehouse.php`

```php
class Warehouse extends Model {
    use SoftDeletes, HasFactory;
    protected $fillable = ['code', 'name', 'notes'];
    public function stockQuantities() { return $this->hasMany(StockQuantity::class); }
    public function shipments()       { return $this->hasMany(Shipment::class); }
}
```

### 2. WarehouseController 🔵

**信頼性**: 🔵 *Webルート仕様より*

**実装ファイル**: `app/Http/Controllers/WarehouseController.php`

- `index()`: 一覧（在庫数サマリー付き）
- `store()`: バリデーション（code 一意, name 必須）
- `update()`: バリデーション同上
- `destroy()`: SoftDelete

### 3. ルート設定 🔵

**信頼性**: 🔵 *Webルート仕様より*

```php
Route::middleware(['auth', 'role:admin'])->group(function () {
    Route::resource('warehouses', WarehouseController::class)->except(['show']);
});
```
