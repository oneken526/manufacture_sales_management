## タスク概要
見積書の一覧表示と詳細表示を担う `QuotationController` の `index` / `show` アクション、ルート設定、および対応する Blade ビューを実装する。ステータスフィルタ・得意先検索・ページネーションを備えた一覧、明細・承認状況を表示する詳細画面を提供する。

## 依存タスク
- **前提タスク**: TASK-0013（QuotationService）, TASK-0004（レイアウト基盤）
- **後続タスク**: TASK-0016（見積承認ワークフロー）

## 完了条件
- [ ] `GET /quotations` で見積一覧が表示される（admin / sales のみ）

- [ ] `GET /quotations/{id}` で見積詳細（明細・承認状況）が表示される

## 実装詳細

### 1. QuotationController（index / show）🔵
**信頼性**: 🔵 *REQ-013・REQ-015 より*

**実装ファイル**: `app/Http/Controllers/QuotationController.php`

#### `index(Request $request): View`
- クエリパラメータでフィルタリングを行う:
  - `status`: `draft` / `pending` / `approved` / `rejected`
  - `customer_name`: 得意先名の部分一致検索
- `Quotation::with(['customer', 'user'])` でリレーションを Eager Loading する
- `paginate(20)` でページネーションを適用する
- ロール制限: `admin` / `sales`

#### `show(Quotation $quotation): View`
- `Quotation::with(['customer', 'user', 'approvedBy', 'details.product'])` でリレーションを取得する
- admin は全件閲覧可能、sales は自分が作成した見積のみ閲覧可（ポリシーで制御）🟡
  - **信頼性**: 🟡 *閲覧権限の詳細仕様が要件定義書で明示されていないため要確認*
- PDF リンクを表示する（TASK-0016 以降で実装）

---

### 2. ルート設定 🔵
**信頼性**: 🔵 *Webルート仕様より*

**実装ファイル**: `routes/web.php`

```php
Route::middleware(['auth', 'role:admin|sales'])->group(function () {
    Route::resource('quotations', QuotationController::class)
        ->only(['index', 'show', 'create', 'store', 'edit', 'update', 'destroy']);

    // 承認ワークフロー（TASK-0016 で追加）
    Route::post('quotations/{quotation}/submit',    [QuotationController::class, 'submit']);
    Route::post('quotations/{quotation}/approve',   [QuotationController::class, 'approve'])
        ->middleware('role:admin');
    Route::post('quotations/{quotation}/reject',    [QuotationController::class, 'reject'])
        ->middleware('role:admin');
    Route::post('quotations/{quotation}/duplicate', [QuotationController::class, 'duplicate']);
    Route::post('quotations/{quotation}/convert',   [QuotationController::class, 'convertToOrder']);
});
```

### 3. Blade ビュー（index）🔵
**信頼性**: 🔵 *UI仕様より*

**実装ファイル**: `resources/views/quotations/index.blade.php`

- `@extends('layouts.app')` を使用する
- 検索フォーム（ステータスプルダウン・得意先名テキストボックス・検索ボタン）
- 見積一覧テーブル:
  - 見積番号 / 得意先名 / 作成者 / 有効期限 / ステータスバッジ / 操作ボタン
- ステータスバッジ: ステータスごとに色分けする（draft: グレー / pending: 黄 / approved: 緑 / rejected: 赤）
- ページネーションリンク（`$quotations->links()`）
- 「新規作成」ボタン（`/quotations/create` へリンク）

---

### 4. Blade ビュー（show）🔵
**信頼性**: 🔵 *UI仕様より*

**実装ファイル**: `resources/views/quotations/show.blade.php`

- 見積書ヘッダー情報（見積番号・得意先・有効期限・ステータス・申請日・承認日）
- 明細テーブル（商品名・数量・単価・金額）
- 合計金額の表示
- 承認状況の表示（承認者名・承認日時）
- 却下理由の表示（rejected の場合のみ）
- 操作ボタン（ステータスに応じて表示を切り替える）:
  - `draft`: 編集ボタン / 承認申請ボタン
  - `pending`: 承認ボタン（admin のみ）/ 却下ボタン（admin のみ）
  - `approved`: 受注変換ボタン / 複製ボタン
  - `rejected`: 複製ボタン
