## タスク概要

見積書の新規作成・編集フォームを実装する。明細行は jQuery で動的に追加・削除でき、商品選択時に Ajax で得意先別単価を自動取得して単価欄に反映する（REQ-020）。小計・合計・消費税を自動計算してリアルタイムに表示する。フォーム送信時は `QuotationController::store()` / `update()` 経由で `QuotationService` を呼ぶ。

## 依存タスク

- **前提タスク**: TASK-0013（QuotationService）, TASK-0004（レイアウト）, TASK-0010（特別単価 Ajax）
- **後続タスク**: TASK-0016（見積承認ワークフロー）

## 実装詳細

### 1. QuotationController::create / store / edit / update 🔵

**信頼性**: 🔵 *Webルート仕様より*

**実装ファイル**: `app/Http/Controllers/QuotationController.php`

```php
public function store(QuotationRequest $request) {
    $quotation = $this->quotationService->create(
        auth()->user(),
        $request->validated()
    );
    return redirect()->route('quotations.show', $quotation)
        ->with('success', '見積書を作成しました');
}
```

### 2. Blade ビュー（明細行動的追加） 🔵

### 3. JavaScript（unit_price Ajax + 合計自動計算） 🔵

**信頼性**: 🔵 *要件定義 REQ-020 より*

**実装ファイル**: `resources/js/quotation-form.js`

```javascript
$(document).on('change', '.product-select', function() {
    const customerId = $('#customer_id').val();
    const productId  = $(this).val();
    const $row = $(this).closest('tr');
    $.getJSON(`/api/customers/${customerId}/unit-price`, { product_id: productId }, (data) => {
        $row.find('.unit-price').val(data.unit_price);
        recalcRow($row);
        recalcTotal();
    });
});
```

### 4. バリデーション 🔵

**信頼性**: 🔵 *DBスキーマ・要件定義より*

**実装ファイル**: `app/Http/Requests/QuotationRequest.php`

- `customer_id`: required, exists:customers,id
- `delivery_date`: nullable, date, after:today
- `details`: required, array, min:1
- `details.*.product_id`: required, exists:products,id
- `details.*.quantity`: required, integer, min:1
- `details.*.unit_price`: required, numeric, min:0
