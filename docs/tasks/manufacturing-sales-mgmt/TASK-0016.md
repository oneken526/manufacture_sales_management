## タスク概要

見積書の承認ワークフロー（申請→承認/却下）を実装する。申請時は `QuotationService::requestApproval()` が呼ばれ、ステータスを `draft → pending` に遷移させ、admin へ承認申請メールを送信する（REQ-021）。承認・却下は admin のみ可能。承認後は `approved`、却下後は `rejected` に遷移し、申請者へ結果メールを送信する（REQ-022）。メール送信失敗時はロールバック（EDGE-001）。

## 実装詳細

### 1. QuotationService::requestApproval 🔵

**信頼性**: 🔵 *要件定義 REQ-015・EDGE-001・dataflow.md より*

**実装ファイル**: `app/Services/QuotationService.php`

```php
public function requestApproval(Quotation $quotation, User $actor): void {
    if ($quotation->status !== 'draft') {
        throw new InvalidStatusTransitionException('申請可能な状態ではありません');
    }
    DB::transaction(function () use ($quotation, $actor) {
        $quotation->update(['status' => 'pending', 'requested_by' => $actor->id]);
        $admins = User::role('admin')->get();
        foreach ($admins as $admin) {
            Mail::to($admin->email)->send(new ApprovalRequestMail($quotation, $actor));
        }
    });
}
```

### 2. QuotationService::approve / reject 🔵

**信頼性**: 🔵 *要件定義 REQ-015・EDGE-001・dataflow.md より*

```php
public function approve(Quotation $quotation, User $approver): void {
    if ($quotation->status !== 'pending') {
        throw new InvalidStatusTransitionException('承認可能な状態ではありません');
    }
    DB::transaction(function () use ($quotation, $approver) {
        $quotation->update(['status' => 'approved', 'approved_by' => $approver->id,
                            'approved_at' => now()]);
        $requester = User::find($quotation->requested_by);
        Mail::to($requester->email)->send(new ApprovalResultMail($quotation, 'approved'));
    });
}
```

### 3. QuotationController（承認アクション） 🔵

**信頼性**: 🔵 *Webルート仕様より*

**実装ファイル**: `app/Http/Controllers/QuotationController.php`（追記）

```php
public function approve(Quotation $quotation) {
    $this->authorize('approve', $quotation); // admin のみ
    $this->quotationService->approve($quotation, auth()->user());
    return back()->with('success', '承認しました');
}
```

## 単体テスト要件

### テストケース1: 承認申請でステータス遷移とメール送信 🔵

**Given**: status=draft の見積、sales ユーザー
**When**: `QuotationService::requestApproval()` を呼ぶ
**Then**: status が pending に変わり、admin にメールが送信される

### テストケース2: メール失敗でロールバック（EDGE-001） 🔵

**Given**: status=draft の見積
**When**: Mail::send が例外をスローする
**Then**: status は draft のまま、DB 変更がロールバックされる

### テストケース3: sales が承認しようとすると 403 🔵

**Given**: sales ユーザー、status=pending の見積
**When**: POST /quotations/{id}/approve
**Then**: 403 Forbidden が返る
