## 実装詳細

### 1. QuotationService::convertToOrder 🔵

**信頼性**: 🔵 *要件定義 REQ-018・dataflow.md より*

**実装ファイル**: `app/Services/QuotationService.php`（追記）

```php
public function convertToOrder(Quotation $quotation, User $actor): Order {
    if ($quotation->status !== 'approved') {
        throw new InvalidStatusTransitionException('承認済み見積のみ受注に変換できます');
    }
    return DB::transaction(function () use ($quotation, $actor) {
        $order = $this->orderService->createFromQuotation($quotation, $actor);
        $quotation->update(['status' => 'converted', 'converted_order_id' => $order->id]);
        return $order;
    });
}
```

### 2. QuotationService::duplicate 🔵

**信頼性**: 🔵 *要件定義 REQ-013 より*

```php
public function duplicate(Quotation $quotation, User $actor): Quotation {
    return DB::transaction(function () use ($quotation, $actor) {
        $newQuotation = $quotation->replicate(['status', 'quotation_number',
            'approved_by', 'approved_at', 'requested_by']);
        $newQuotation->status           = 'draft';
        $newQuotation->quotation_number = $this->numberingService->generate('Q');
        $newQuotation->created_by       = $actor->id;
        $newQuotation->save();
        foreach ($quotation->details as $detail) {
            $newDetail = $detail->replicate();
            $newDetail->quotation_id = $newQuotation->id;
            $newDetail->save();
        }
        return $newQuotation;
    });
}
```

### 3. 見積書 PDF 出力 🔵

**信頼性**: 🔵 *要件定義 REQ-046・REQ-052 より*

**実装ファイル**: `app/Http/Controllers/QuotationController.php`（追記）

```php
public function pdf(Quotation $quotation) {
    $pdf = \Barryvdh\DomPDF\Facade\Pdf::loadView('pdf.quotation', compact('quotation'));
    $pdf->setPaper('A4');
    return $pdf->download("quotation-{$quotation->quotation_number}.pdf");
}
```

**Blade テンプレート**: `resources/views/pdf/quotation.blade.php`
- 日本語フォント指定（IPAex ゴシックまたは Noto Sans JP）
