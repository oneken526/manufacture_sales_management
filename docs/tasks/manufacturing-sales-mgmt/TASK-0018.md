## 実装詳細

### 1. Order モデル 🔵

**信頼性**: 🔵 *DBスキーマ・要件定義 REQ-016 より*

**実装ファイル**: `app/Models/Order.php`

```php
class Order extends Model {
    use SoftDeletes, HasFactory;
    protected $fillable = [
        'customer_id', 'quotation_id', 'order_number', 'status',
        'order_date', 'delivery_date', 'notes',
        'requested_by', 'approved_by', 'approved_at',
    ];
    public function details()           { return $this->hasMany(OrderDetail::class); }
    public function customer()          { return $this->belongsTo(Customer::class); }
    public function quotation()         { return $this->belongsTo(Quotation::class); }
    public function manufactureOrders() { return $this->hasMany(ManufactureOrder::class); }
}
```

    protected $fillable = [
        'order_id', 'product_id', 'quantity', 'unit_price', 'shipped_quantity',

    public function remainingQuantity(): int {
        return $this->quantity - $this->shipped_quantity;
    }
}

### 3. OrderService::create 🔵

**信頼性**: 🔵 *要件定義 REQ-016・REQ-024・dataflow.md より*

**実装ファイル**: `app/Services/OrderService.php`

```php
public function create(array $data, User $actor): array {
    $customer = Customer::findOrFail($data['customer_id']);
    $creditExceeded = $this->checkCredit($customer, $data);

    $order = DB::transaction(function () use ($data, $actor) {
        $order = Order::create(array_merge($data, [
            'order_number' => $this->numberingService->generate('O'),
            'status'       => 'draft',
            'created_by'   => $actor->id,
        ]));
        foreach ($data['details'] as $detail) {
            $order->details()->create($detail);
        }
        return $order;
    });

    return ['order' => $order, 'credit_exceeded' => $creditExceeded];
}
```

### 4. OrderService::approve（製造指示自動発行） 🔵

**信頼性**: 🔵 *要件定義 REQ-023・EDGE-007・dataflow.md より*

```php
public function approve(Order $order, User $approver): void {
    if ($order->status !== 'pending') {
        throw new InvalidStatusTransitionException('承認可能な状態ではありません');
    }
    DB::transaction(function () use ($order, $approver) {
        $order->update(['status' => 'approved',
                        'approved_by' => $approver->id, 'approved_at' => now()]);
        // 製造指示自動発行 (EDGE-007: 失敗でロールバック)
        $this->manufactureOrderService->createFromOrder($order, $approver);
        $requester = User::find($order->requested_by);
        Mail::to($requester->email)->send(new ApprovalResultMail($order, 'approved'));
    });
}
```
