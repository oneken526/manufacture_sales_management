## タスク概要

受注管理の Web 画面（一覧・詳細・作成・編集）を実装する。一覧は受注残（未出荷数量）・納期・ステータスフィルタを提供し、詳細は明細ごとの受注残を表示する（REQ-019）。登録・編集フォームは見積作成フォームと同様に jQuery で明細行を動的追加し、得意先別単価 Ajax を使用する。与信超過時はフォーム上に警告バナーを表示する（REQ-024）。

## 実装詳細

### 1. OrderController 🔵

**信頼性**: 🔵 *Webルート仕様より*

**実装ファイル**: `app/Http/Controllers/OrderController.php`

```php
public function store(OrderRequest $request) {
    $result = $this->orderService->create($request->validated(), auth()->user());
    $order  = $result['order'];

    if ($result['credit_exceeded']) {
        session()->flash('warning', '与信限度額を超過しています（受注は登録されました）');
    }
    return redirect()->route('orders.show', $order)->with('success', '受注を登録しました');
}
```
