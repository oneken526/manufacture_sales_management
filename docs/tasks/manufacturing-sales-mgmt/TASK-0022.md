## タスク概要
製造指示の一覧・詳細表示とステータス更新（開始・完了・キャンセル）を提供する Web 画面を実装する。詳細画面では製番トレース（受注 → 製造 → 在庫ロット → 出荷の連鎖）を表示する（US-024）。画面へのアクセスは manufacture / admin ロールに限定し、キャンセルは admin のみとする。

## 依存タスク
- **前提タスク**: TASK-0021
- **後続タスク**: TASK-0024

## 実装詳細

### 1. ManufactureOrderController 🔵
**信頼性**: 🔵 *REQ-027, US-021, US-022, US-024 より*

**ファイル**: `app/Http/Controllers/ManufactureOrderController.php`

#### index 🔵
**信頼性**: 🔵 *US-021（製造指示一覧）より*

- `manufacture_orders` を `status` クエリパラメータでフィルタする（未指定時は全件）
- 各行に製番（manufacture_number）、関連受注の納期（order.delivery_date）、ステータスバッジを表示する
- ページネーションを適用する

#### show 🔵
**信頼性**: 🔵 *US-024（製番トレース）より*

- ManufactureOrder に紐づく以下の情報を eager loading して渡す
  - 関連受注（order）・受注明細（order.details）
  - 在庫ロット（stockLots）・倉庫別数量（stockLots.stockQuantities）
  - 出荷（shipments）・出荷明細（shipments.details）
- Blade ビューで受注 → 製造 → 在庫ロット → 出荷の連鎖をツリー表示する

#### start 🔵
**信頼性**: 🔵 *REQ-027（着手操作）より*

- `POST /manufacture-orders/{id}/start`
- `ManufactureOrderService::start($mo)` を呼び出す
- 成功後は詳細画面（show）にリダイレクトし、フラッシュメッセージを表示する
- ロール: manufacture, admin

#### complete 🔵
**信頼性**: 🔵 *REQ-028（完了操作）より*

- `POST /manufacture-orders/{id}/complete`
- 完了フォームのデータを `ManufactureOrderService::complete()` に渡す
- 詳細は TASK-0024（製造完了フォーム）で実装する

#### cancel 🔵
**信頼性**: 🔵 *REQ-028（admin のみキャンセル可）より*

- `POST /manufacture-orders/{id}/cancel`
- `ManufactureOrderService::cancel($mo, auth()->user())` を呼び出す
- ロール: admin のみ（ミドルウェアまたは Gate で制御）
- 成功後は一覧画面にリダイレクトし、フラッシュメッセージを表示する

### 2. Blade ビュー 🔵
**信頼性**: 🔵 *US-021, US-022, US-024 より*

#### manufacture_orders/index.blade.php 🔵
- ステータスフィルタタブ（全て / pending / in_progress / completed / cancelled）
- テーブルカラム: 製番、商品名、数量、納期、ステータスバッジ、操作ボタン
- ステータスバッジ: Tailwind CSS でカラーコーディング（pending=gray, in_progress=blue, completed=green, cancelled=red）
- 操作ボタン: 詳細・開始・完了・キャンセル（ロールと状態に応じて表示制御）

#### manufacture_orders/show.blade.php 🔵
- 製番・ステータス・開始日時・完了日時を表示する
- 関連受注リンク・受注明細テーブルを表示する
- **製番トレースセクション（US-024）**: 受注 → 製造 → 在庫ロット → 出荷の連鎖をツリー形式で表示する
- 操作ボタン: 開始・完了・キャンセル（ロールと状態に応じて表示制御）

**実装ファイル**:
- `resources/views/manufacture_orders/index.blade.php`
- `resources/views/manufacture_orders/show.blade.php`

### 3. ルート定義 🔵
**信頼性**: 🔵 *api-endpoints.md より*

```php
// routes/web.php
Route::middleware(['auth', 'role:manufacture|admin'])->group(function () {
    Route::get('/manufacture-orders', [ManufactureOrderController::class, 'index'])->name('manufacture-orders.index');
    Route::get('/manufacture-orders/{id}', [ManufactureOrderController::class, 'show'])->name('manufacture-orders.show');
    Route::post('/manufacture-orders/{id}/start', [ManufactureOrderController::class, 'start'])->name('manufacture-orders.start');
    Route::get('/manufacture-orders/{id}/complete', [ManufactureOrderController::class, 'completeForm'])->name('manufacture-orders.complete-form');
    Route::post('/manufacture-orders/{id}/complete', [ManufactureOrderController::class, 'complete'])->name('manufacture-orders.complete');
});

Route::middleware(['auth', 'role:admin'])->group(function () {
    Route::post('/manufacture-orders/{id}/cancel', [ManufactureOrderController::class, 'cancel'])->name('manufacture-orders.cancel');
});
```
