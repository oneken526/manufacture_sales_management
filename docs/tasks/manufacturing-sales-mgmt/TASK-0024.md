## 実装詳細

### 1. StockController 🔵

**信頼性**: 🔵 *Webルート仕様より*

**実装ファイル**: `app/Http/Controllers/StockController.php`

```php
public function index(Request $request) {
    $query = StockLot::with(['product', 'stockQuantities.warehouse', 'manufactureOrder'])
        ->withSum('stockQuantities', 'quantity');

    if ($request->product_id) {
        $query->where('product_id', $request->product_id);
    }
    if ($request->warehouse_id) {
        $query->whereHas('stockQuantities',
            fn($q) => $q->where('warehouse_id', $request->warehouse_id)->where('quantity', '>', 0));
    }
    if ($request->lot_number) {
        $query->where('lot_number', 'like', "%{$request->lot_number}%");
    }

    $lots = $query->latest()->paginate(30);
    return view('stocks.index', compact('lots'));
}
```
