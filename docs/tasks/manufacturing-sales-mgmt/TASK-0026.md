## タスク概要

棚卸時に実在庫数量と帳簿在庫数量の差異を調整する画面を実装する。ロット・倉庫を指定して実在庫数量を入力し、`StockService::adjust()` を呼んで差分を `stock_movements`（type=adjustment）に記録する。アクセスは admin / warehouse ロールに限定する（REQ-032）。

## 実装詳細

### 1. StockService::adjust 🔵

**信頼性**: 🔵 *要件定義 REQ-032 より*

**実装ファイル**: `app/Services/StockService.php`（追記）

```php
public function adjust(StockLot $lot, int $warehouseId, int $actualQty, ?string $notes = null): void {
    DB::transaction(function () use ($lot, $warehouseId, $actualQty, $notes) {
        $sq = StockQuantity::where(['lot_id' => $lot->id, 'warehouse_id' => $warehouseId])
            ->lockForUpdate()->firstOrFail();
        $diff = $actualQty - $sq->quantity;
        $sq->update(['quantity' => $actualQty]);
        StockMovement::create([
            'lot_id'       => $lot->id,
            'warehouse_id' => $warehouseId,
            'type'         => 'adjustment',
            'quantity'     => $diff,
            'notes'        => $notes,
        ]);
    });
}
```
