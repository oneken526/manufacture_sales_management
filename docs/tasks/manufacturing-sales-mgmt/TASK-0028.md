## タスク概要

出荷の登録・一覧・詳細画面を実装する。登録フォームでは受注を選択すると受注残明細が表示され、各明細に対してロットを手動選択（Ajax でロット在庫一覧を取得）し、出荷数量を入力する（REQ-038）。アクセス権限は admin / warehouse / sales（閲覧のみ）。

- **前提タスク**: TASK-0027（ShipmentService）, TASK-0004（レイアウト）
- **後続タスク**: TASK-0029（出荷確定・返品）

### 1. ShipmentController 🔵

**信頼性**: 🔵 *Webルート仕様より*

**実装ファイル**: `app/Http/Controllers/ShipmentController.php`

```php
public function store(ShipmentRequest $request) {
    $order = Order::findOrFail($request->order_id);
    $shipment = $this->shipmentService->create(
        $order,
        $request->warehouse_id,
        $request->details,
        auth()->user()
    );
    return redirect()->route('shipments.show', $shipment)
        ->with('success', '出荷を登録しました');
}
```

### 2. 受注残明細 Ajax エンドポイント 🔵

**信頼性**: 🔵 *Webルート仕様・要件定義 REQ-038 より*

```php
// GET /api/orders/{order}/remaining-details
public function remainingDetails(Order $order) {
    $details = $order->details()
        ->with('product')
        ->whereRaw('quantity > shipped_quantity')
        ->get()
        ->map(fn($d) => [
            'id'          => $d->id,
            'product_id'  => $d->product_id,
            'product_name'=> $d->product->name,
            'remaining'   => $d->remainingQuantity(),
        ]);
    return response()->json($details);
}
```

### 3. ロット在庫 Ajax エンドポイント 🔵

**信頼性**: 🔵 *要件定義 REQ-038 より*

```php
// GET /api/products/{product}/stock-lots?warehouse_id={id}
public function stockLots(Product $product, Request $request) {
    $lots = $product->stockLots()
        ->with(['stockQuantities' => fn($q) => $q->where('warehouse_id', $request->warehouse_id)])
        ->whereHas('stockQuantities', fn($q) =>
            $q->where('warehouse_id', $request->warehouse_id)->where('quantity', '>', 0))
        ->get()
        ->map(fn($l) => [
            'id'         => $l->id,
            'lot_number' => $l->lot_number,
            'quantity'   => $l->stockQuantities->first()?->quantity ?? 0,
        ]);
    return response()->json($lots);
}
```
