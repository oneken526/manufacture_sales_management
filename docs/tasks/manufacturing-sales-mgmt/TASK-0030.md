### 1. Invoice モデル 🔵
**信頼性**: 🔵 *DBスキーマ・設計文書より*

**実装ファイル**: `app/Models/Invoice.php`

- `SoftDeletes` トレイトを使用する
- `fillable`:
  - `invoice_number`, `customer_id`, `invoice_date`, `closing_date`
  - `billing_period_start`, `billing_period_end`
  - `subtotal`, `tax_rate`, `tax_amount`, `total_amount`, `remaining_amount`
  - `status`, `created_by`
- リレーション:
  - `belongsTo(Customer::class)` — 請求先得意先
  - `belongsTo(User::class, 'created_by')` — 作成者
  - `hasMany(InvoiceDetail::class)` — 請求明細
  - `hasMany(PaymentAllocation::class)` — 入金消込

```php
<?php

declare(strict_types=1);

namespace App\Models;

use Illuminate\Database\Eloquent\Factories\HasFactory;
use Illuminate\Database\Eloquent\Model;
use Illuminate\Database\Eloquent\Relations\BelongsTo;
use Illuminate\Database\Eloquent\Relations\HasMany;
use Illuminate\Database\Eloquent\SoftDeletes;

class Invoice extends Model
{
    use HasFactory, SoftDeletes;

    protected $fillable = [
        'invoice_number',
        'customer_id',
        'invoice_date',
        'closing_date',
        'billing_period_start',
        'billing_period_end',
        'subtotal',
        'tax_rate',
        'tax_amount',
        'total_amount',
        'remaining_amount',
        'status',
        'created_by',
    ];

    protected $casts = [
        'invoice_date'         => 'date',
        'closing_date'         => 'date',
        'billing_period_start' => 'date',
        'billing_period_end'   => 'date',
        'subtotal'             => 'integer',
        'tax_rate'             => 'decimal:2',
        'tax_amount'           => 'integer',
        'total_amount'         => 'integer',
        'remaining_amount'     => 'integer',
    ];

    public function customer(): BelongsTo
    {
        return $this->belongsTo(Customer::class);
    }

    public function createdBy(): BelongsTo
    {
        return $this->belongsTo(User::class, 'created_by');
    }

    public function details(): HasMany
    {
        return $this->hasMany(InvoiceDetail::class);
    }

    public function paymentAllocations(): HasMany
    {
        return $this->hasMany(PaymentAllocation::class);
    }
}
```

### 3. InvoiceService 🔵
**信頼性**: 🔵 *要件定義書 REQ-040・締日処理仕様より*

**実装ファイル**: `app/Services/InvoiceService.php`

#### `generate(Customer $customer, int $year, int $month): Invoice`

**締日計算ロジック**:
- `closing_day = 99` → 月末（`Carbon::create($year, $month)->endOfMonth()`）
- `closing_day = N (1〜28)` → `$year年$month月N日`
- 請求期間（billing_period）:
  - 前月締日の翌日〜当月締日

**処理フロー**:
1. 当月締日・前月締日を計算し、billing_period_start / billing_period_end を確定する
2. EDGE-004: 同一得意先・同一締め月（closing_date が同月）の既存請求書を確認し、存在する場合は `DuplicateInvoiceException` をスローする
3. `NumberingService::generate('INV')` で invoice_number を採番する
4. 対象期間内の確定出荷（`shipments.status = 'confirmed'`、`customer_id` 一致）を取得する
5. `DB::transaction()` 内で `invoices` レコードを作成し、対象出荷ごとに `invoice_details` を挿入する
6. 消費税率 10%（`tax_rate = 0.10`）を適用し、`subtotal`・`tax_amount`・`total_amount` を計算する
7. `remaining_amount = total_amount`（初期値）を設定する

```php
<?php

declare(strict_types=1);

namespace App\Services;

use App\Exceptions\DuplicateInvoiceException;
use App\Models\Customer;
use App\Models\Invoice;
use App\Models\InvoiceDetail;
use Carbon\Carbon;
use Illuminate\Support\Facades\DB;

class InvoiceService
{
    public function __construct(
        private readonly NumberingService $numberingService,
    ) {}

    /**
     * 得意先・年月を指定して月次請求書を生成する。
     *
     * @throws DuplicateInvoiceException 同一得意先・同一締め月の請求書が既に存在する場合
     */
    public function generate(Customer $customer, int $year, int $month): Invoice
    {
        $closingDate  = $this->resolveClosingDate($customer->closing_day, $year, $month);
        $prevClosing  = $this->resolveClosingDate($customer->closing_day, ...$this->prevYearMonth($year, $month));
        $periodStart  = $prevClosing->copy()->addDay();
        $periodEnd    = $closingDate->copy();

        // EDGE-004: 重複チェック
        $exists = Invoice::withoutTrashed()
            ->where('customer_id', $customer->id)
            ->whereYear('closing_date', $closingDate->year)
            ->whereMonth('closing_date', $closingDate->month)
            ->exists();

        if ($exists) {
            throw new DuplicateInvoiceException(
                "得意先ID {$customer->id} の {$year}年{$month}月請求書は既に存在します。"
            );
        }

        return DB::transaction(function () use ($customer, $year, $month, $closingDate, $periodStart, $periodEnd): Invoice {
            $invoiceNumber = $this->numberingService->generate('INV');

            $shipments = \App\Models\Shipment::where('customer_id', $customer->id)
                ->where('status', 'confirmed')
                ->whereBetween('shipped_at', [$periodStart->startOfDay(), $periodEnd->endOfDay()])
                ->with('details.product')
                ->get();

            $subtotal = 0;
            $details  = [];

            foreach ($shipments as $shipment) {
                foreach ($shipment->details as $detail) {
                    $amount    = $detail->quantity * $detail->unit_price;
                    $subtotal += $amount;
                    $details[] = [
                        'shipment_id' => $shipment->id,
                        'product_id'  => $detail->product_id,
                        'quantity'    => $detail->quantity,
                        'unit_price'  => $detail->unit_price,
                        'amount'      => $amount,
                    ];
                }
            }

            $taxRate   = 0.10;
            $taxAmount = (int) round($subtotal * $taxRate);
            $total     = $subtotal + $taxAmount;

            $invoice = Invoice::create([
                'invoice_number'      => $invoiceNumber,
                'customer_id'         => $customer->id,
                'invoice_date'        => now()->toDateString(),
                'closing_date'        => $closingDate->toDateString(),
                'billing_period_start'=> $periodStart->toDateString(),
                'billing_period_end'  => $periodEnd->toDateString(),
                'subtotal'            => $subtotal,
                'tax_rate'            => $taxRate,
                'tax_amount'          => $taxAmount,
                'total_amount'        => $total,
                'remaining_amount'    => $total,
                'status'              => 'issued',
                'created_by'          => auth()->id(),
            ]);

            foreach ($details as $detail) {
                InvoiceDetail::create(array_merge($detail, ['invoice_id' => $invoice->id]));
            }

            return $invoice->fresh('details');
        });
    }

    /** 締日を Carbon インスタンスで返す */
    private function resolveClosingDate(int $closingDay, int $year, int $month): Carbon
    {
        if ($closingDay === 99) {
            return Carbon::create($year, $month)->endOfMonth()->startOfDay();
        }

        return Carbon::create($year, $month, min($closingDay, Carbon::create($year, $month)->daysInMonth))->startOfDay();
    }

    /** 前月の年月を返す */
    private function prevYearMonth(int $year, int $month): array
    {
        $dt = Carbon::create($year, $month)->subMonth();
        return [$dt->year, $dt->month];
    }
}
```

### テストケース1: 基本請求書生成 🔵
**信頼性**: 🔵 *REQ-040 月次請求書生成要件より*
**Given**: 得意先（closing_day=20）、対象月の確定出荷が2件存在する
**When**: `InvoiceService::generate($customer, $year, $month)` を呼び出す
**Then**:
- invoices テーブルに1件レコードが作成される
- invoice_number が `INV{YYYYMM}-001` 形式である
- billing_period_start / billing_period_end が正しく設定される
- subtotal・tax_amount・total_amount が正しく計算される
- remaining_amount が total_amount と等しい

**When**: `generate($customer, 2026, 4)` を呼び出す
**Then**: closing_date が 2026-04-30 になる

### テストケース3: 締日 closing_day=15（15日締め） 🔵
