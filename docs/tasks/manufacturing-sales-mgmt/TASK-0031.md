## タスク概要

請求書の一覧・詳細・月次生成（手動実行）・売掛金元帳画面を実装する。月次請求書生成は admin が対象年月と得意先（全件 or 指定）を選択して手動実行し（REQ-045）、`InvoiceService::generate()` を呼ぶ。売掛金元帳は得意先別の請求・入金・残高を時系列で表示する（REQ-041）。

## 実装詳細

### 1. InvoiceController 🔵

**信頼性**: 🔵 *Webルート仕様より*

**実装ファイル**: `app/Http/Controllers/InvoiceController.php`

```php
public function generate(GenerateInvoiceRequest $request) {
    $year  = $request->year;
    $month = $request->month;
    $customers = $request->customer_id
        ? Customer::where('id', $request->customer_id)->get()
        : Customer::all();

    $generated = 0;
    $errors    = [];
    foreach ($customers as $customer) {
        try {
            $this->invoiceService->generate($customer, $year, $month);
            $generated++;
        } catch (DuplicateInvoiceException $e) {
            $errors[] = "{$customer->name}: 既に生成済みです（EDGE-004）";
        }
    }
    return back()->with('success', "{$generated}件の請求書を生成しました")
                 ->with('errors', $errors);
}
```

### 2. 売掛金元帳 🔵

**信頼性**: 🔵 *要件定義 REQ-041 より*

```php
public function ledger(Customer $customer) {
    $invoices = $customer->invoices()
        ->with(['payments' => fn($q) => $q->orderBy('payment_date')])
        ->orderBy('invoice_date')
        ->get();
    return view('invoices.ledger', compact('customer', 'invoices'));
}
```
