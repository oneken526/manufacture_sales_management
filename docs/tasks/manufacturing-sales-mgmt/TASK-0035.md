## 実装詳細

### 1. Blade テンプレート構造 🔵

**信頼性**: 🔵 *要件定義 REQ-046 より*

**実装ファイル**: `resources/views/pdf/quotation.blade.php`

```blade
<!DOCTYPE html>
<html lang="ja">
<head>
    <meta charset="UTF-8">
    <style>
        body { font-family: 'ipaexg'; font-size: 10pt; }
        table { width: 100%; border-collapse: collapse; }
        th, td { border: 1px solid #333; padding: 4px; }
        .header { text-align: right; margin-bottom: 20px; }
        .total-area { text-align: right; margin-top: 10px; }
    </style>
</head>
<body>
    <h2 style="text-align: center;">見 積 書</h2>
    <div class="header">
        <p>見積番号: {{ $quotation->quotation_number }}</p>
        <p>発行日: {{ $quotation->created_at->format('Y年m月d日') }}</p>
        <p>有効期限: {{ $quotation->valid_until?->format('Y年m月d日') ?? '—' }}</p>
    </div>
    <p>{{ $quotation->customer->name }} 御中</p>
    <table>
        <thead>
            <tr><th>商品名</th><th>数量</th><th>単価</th><th>金額</th></tr>
        </thead>
        <tbody>
            @foreach($quotation->details as $detail)
            <tr>
                <td>{{ $detail->product->name }}</td>
                <td style="text-align:right;">{{ number_format($detail->quantity) }}</td>
                <td style="text-align:right;">{{ number_format($detail->unit_price) }}</td>
                <td style="text-align:right;">{{ number_format($detail->quantity * $detail->unit_price) }}</td>
            </tr>
            @endforeach
        </tbody>
    </table>
    <div class="total-area">
        <p>小計: {{ number_format($quotation->subtotal) }} 円</p>
        <p>消費税(10%): {{ number_format($quotation->tax_amount) }} 円</p>
        <p><strong>合計: {{ number_format($quotation->total_amount) }} 円</strong></p>
    </div>
</body>
</html>
```
