## 実装詳細

### 1. Blade テンプレート 🔵

**信頼性**: 🔵 *要件定義 REQ-047 より*

**実装ファイル**: `resources/views/pdf/delivery.blade.php`

```blade
<!DOCTYPE html>
<html lang="ja">
<head>
    <meta charset="UTF-8">
    <style>
        body { font-family: 'ipaexg'; font-size: 10pt; }
        table { width: 100%; border-collapse: collapse; }
        th, td { border: 1px solid #333; padding: 4px; }
        .receipt-box { border: 2px solid #333; width: 200px; height: 80px;
                       margin-top: 40px; padding: 5px; }
    </style>
</head>
<body>
    <h2 style="text-align: center;">納 品 書</h2>
    <table style="border: none; margin-bottom: 20px;">
        <tr>
            <td style="border: none; width: 50%;">{{ $shipment->order->customer->name }} 御中</td>
            <td style="border: none; text-align: right;">
                <p>出荷番号: {{ $shipment->shipment_number }}</p>
                <p>出荷日: {{ $shipment->shipped_date?->format('Y年m月d日') }}</p>
            </td>
        </tr>
    </table>
    <table>
        <thead>
            <tr><th>商品名</th><th>ロット番号</th><th>数量</th></tr>
        </thead>
        <tbody>
            @foreach($shipment->details as $detail)
            <tr>
                <td>{{ $detail->stockLot->product->name }}</td>
                <td>{{ $detail->stockLot->lot_number }}</td>
                <td style="text-align:right;">{{ number_format($detail->quantity) }}</td>
            </tr>
            @endforeach
        </tbody>
    </table>
    <div style="text-align: right; margin-top: 20px;">
        <div class="receipt-box">受領印</div>
    </div>
</body>
</html>
```
