## 実装詳細

### 1. InvoiceController::pdf 🔵

**信頼性**: 🔵 *Webルート仕様・要件定義 REQ-048 より*

**実装ファイル**: `app/Http/Controllers/InvoiceController.php`（追記）

```php
public function pdf(Invoice $invoice) {
    $invoice->load(['customer', 'details.shipmentDetail.stockLot.product']);
    $pdf = \Barryvdh\DomPDF\Facade\Pdf::loadView('pdf.invoice', compact('invoice'));
    $pdf->setPaper('A4');
    return $pdf->download("invoice-{$invoice->invoice_number}.pdf");
}
```

### 2. Blade テンプレート 🔵

**信頼性**: 🔵 *要件定義 REQ-048 より*

**実装ファイル**: `resources/views/pdf/invoice.blade.php`

```blade
<!DOCTYPE html>
<html lang="ja">
<head>
    <meta charset="UTF-8">
    <style>
        body { font-family: 'ipaexg'; font-size: 10pt; }
        table { width: 100%; border-collapse: collapse; }
        th, td { border: 1px solid #333; padding: 4px; }
    </style>
</head>
<body>
    <h2 style="text-align: center;">請 求 書</h2>
    <p>{{ $invoice->customer->name }} 御中</p>
    <p>請求番号: {{ $invoice->invoice_number }}</p>
    <p>請求日: {{ $invoice->invoice_date->format('Y年m月d日') }}</p>
    <p>支払期限: {{ $invoice->due_date?->format('Y年m月d日') ?? '—' }}</p>

    <table>
        <thead>
            <tr><th>内容</th><th>数量</th><th>単価</th><th>金額</th></tr>
        </thead>
        <tbody>
            @foreach($invoice->details as $detail)
            <tr>
                <td>{{ $detail->description }}</td>
                <td style="text-align:right;">{{ number_format($detail->quantity) }}</td>
                <td style="text-align:right;">{{ number_format($detail->unit_price) }}</td>
                <td style="text-align:right;">{{ number_format($detail->amount) }}</td>
            </tr>
            @endforeach
        </tbody>
    </table>
    <div style="text-align: right; margin-top: 10px;">
        <p>小計: {{ number_format($invoice->subtotal) }} 円</p>
        <p>消費税(10%): {{ number_format($invoice->tax_amount) }} 円</p>
        <p><strong>合計: {{ number_format($invoice->total_amount) }} 円</strong></p>
        <p>残額（未入金）: {{ number_format($invoice->remaining_amount) }} 円</p>
    </div>
</body>
</html>
```
