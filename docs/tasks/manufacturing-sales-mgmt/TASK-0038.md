## 実装詳細

### 1. ReportController::sales 🔵

**信頼性**: 🔵 *Webルート仕様・要件定義 REQ-049 より*

**実装ファイル**: `app/Http/Controllers/ReportController.php`

```php
public function sales(Request $request) {
    $query = ShipmentDetail::query()
        ->join('shipments', 'shipments.id', '=', 'shipment_details.shipment_id')
        ->join('stock_lots', 'stock_lots.id', '=', 'shipment_details.lot_id')
        ->join('products', 'products.id', '=', 'stock_lots.product_id')
        ->join('orders', 'orders.id', '=', 'shipments.order_id')
        ->where('shipments.status', 'confirmed');

    if ($request->year_month) {
        $query->where(DB::raw("DATE_FORMAT(shipments.shipped_date, '%Y-%m')"), $request->year_month);
    }
    if ($request->customer_id) {
        $query->where('orders.customer_id', $request->customer_id);
    }

    $monthly   = (clone $query)->selectRaw("DATE_FORMAT(shipments.shipped_date, '%Y-%m') as month,
                     SUM(shipment_details.quantity * shipment_details.unit_price) as total")
                 ->groupBy('month')->orderBy('month')->get();
    $byCustomer = (clone $query)->selectRaw('orders.customer_id,
                     SUM(shipment_details.quantity * shipment_details.unit_price) as total')
                  ->groupBy('orders.customer_id')->with('order.customer')->get();
    $byProduct  = (clone $query)->selectRaw('products.id as product_id,
                     SUM(shipment_details.quantity) as qty,
                     SUM(shipment_details.quantity * shipment_details.unit_price) as total')
                  ->groupBy('products.id')->get();

    return view('reports.sales', compact('monthly', 'byCustomer', 'byProduct'));
}
```

### 2. CSV エクスポート 🔵

**信頼性**: 🔵 *要件定義 US-041 より*

```php
public function salesCsv(Request $request) {
    $headers = [
        'Content-Type'        => 'text/csv; charset=UTF-8',
        'Content-Disposition' => 'attachment; filename="sales-report.csv"',
    ];
    $callback = function () use ($request) {
        $handle = fopen('php://output', 'w');
        fprintf($handle, chr(0xEF).chr(0xBB).chr(0xBF)); // UTF-8 BOM
        fputcsv($handle, ['年月', '得意先', '商品', '数量', '売上金額']);
        // ... データ出力
        fclose($handle);
    };
    return response()->stream($callback, 200, $headers);
}
```
