### 1. ReportController::stock 🔵

**信頼性**: 🔵 *要件定義 REQ-050 より*

**実装ファイル**: `app/Http/Controllers/ReportController.php`（追記）

```php
public function stock(Request $request) {
    $query = StockMovement::with(['lot.product', 'warehouse'])
        ->orderBy('created_at');

    if ($request->start_date) {
        $query->whereDate('created_at', '>=', $request->start_date);
    }
    if ($request->end_date) {
        $query->whereDate('created_at', '<=', $request->end_date);
    }
    if ($request->product_id) {
        $query->whereHas('lot', fn($q) => $q->where('product_id', $request->product_id));
    }
    if ($request->warehouse_id) {
        $query->where('warehouse_id', $request->warehouse_id);
    }

    $movements = $query->paginate(50);
    $summary = $query->selectRaw('type, SUM(ABS(quantity)) as total_qty, COUNT(*) as cnt')
                     ->groupBy('type')->get();

    return view('reports.stock', compact('movements', 'summary'));
}
```

### 2. CSV エクスポート 🔵

**信頼性**: 🔵 *要件定義 US-041 より*

**実装ファイル**: `app/Http/Controllers/ReportController.php`（追記）

```php
public function stockCsv(Request $request) {
    $headers = [
        'Content-Type'        => 'text/csv; charset=UTF-8',
        'Content-Disposition' => 'attachment; filename="stock-report.csv"',
    ];
    $callback = function () use ($request) {
        $handle = fopen('php://output', 'w');
        fprintf($handle, chr(0xEF).chr(0xBB).chr(0xBF));
        fputcsv($handle, ['日時', '商品名', 'ロット番号', '倉庫', '種別', '数量']);
        // ... データ出力（フィルタ条件を引き継ぐ）
        fclose($handle);
    };
    return response()->stream($callback, 200, $headers);
}
```
