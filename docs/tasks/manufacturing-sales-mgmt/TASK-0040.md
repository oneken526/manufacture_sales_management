## 実装詳細

### 1. ReportController::backorders 🔵

**信頼性**: 🔵 *要件定義 REQ-051 より*

**実装ファイル**: `app/Http/Controllers/ReportController.php`（追記）

```php
public function backorders(Request $request) {
    $query = OrderDetail::query()
        ->with(['order.customer', 'product'])
        ->join('orders', 'orders.id', '=', 'order_details.order_id')
        ->whereColumn('order_details.quantity', '>', 'order_details.shipped_quantity')
        ->where('orders.status', 'approved')
        ->select('order_details.*', 'orders.delivery_date', 'orders.customer_id');

    if ($request->customer_id) {
        $query->where('orders.customer_id', $request->customer_id);
    }
    if ($request->product_id) {
        $query->where('order_details.product_id', $request->product_id);
    }
    if ($request->overdue_only) {
        $query->where('orders.delivery_date', '<', today());
    }

    $details = $query->orderBy('orders.delivery_date')->paginate(50);
    $today   = today();

    return view('reports.backorders', compact('details', 'today'));
}
```
